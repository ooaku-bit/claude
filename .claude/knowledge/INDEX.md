# Knowledge Base Index

組織の知識を体系的に蓄積・検索するためのインデックス。

## カテゴリ

| カテゴリ | 場所 | 用途 |
|----------|------|------|
| 決定記録 (ADR) | `decisions/` | なぜその設計・技術選択をしたかの記録 |
| パターン集 | `patterns/` | 再利用可能なコード・ワークフローパターン |
| 振り返り | `retrospectives/` | セッション・タスク完了後の学び |
| スキル学習 | `skill-learnings/` | スキル使用から得られた改善知見 |

## クイック検索

- 決定記録 → `decisions/` 内の `ADR-NNNN-*.md` ファイル
- パターン → `patterns/PATTERNS.md`
- 最新の振り返り → `retrospectives/` を日付で降順ソート

## 追加方法

- `/knowledge-add` コマンドで即座に追記
- `/retro` コマンドでセッション終了時に振り返りを記録
- スキル使用後は `skill-learnings/` に知見を追記
