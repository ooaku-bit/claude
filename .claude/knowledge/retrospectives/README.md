# Retrospectives

セッション・タスク完了後の振り返り記録。学びを次に活かすための仕組み。

## ファイル命名規則

```
YYYY-MM-DD-<タスク概要>.md
```

## テンプレート

```markdown
# 振り返り: YYYY-MM-DD — <タスク概要>

## やったこと
- 

## うまくいったこと (Keep)
- 

## 改善したいこと (Problem)
- 

## 次回試すこと (Try)
- 

## スキルへのフィードバック
- <スキル名>: 改善提案
```
