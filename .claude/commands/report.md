# /report — 経営レポート作成

月次・週次・案件の経営レポートをエグゼクティブサマリー付きで作成する。

## 使い方

```
/report [レポートの種類: 月次|週次|案件] [データ・情報を貼り付け]
```

## 手順

1. レポート種別と対象期間を確認する
2. 提供されたデータを「実績・課題・アクション」に整理する
3. エグゼクティブサマリーを3行で書く
4. ファイルに保存する

## 出力フォーマット

```
## [種別]レポート  YYYY-MM（または期間）

### エグゼクティブサマリー（3行）
① 実績:
② 課題:
③ アクション:

### 実績・数値
（前期比・目標対比を含む）

### 課題と対策
| 課題 | 重要度 | 対策 | 担当 | 期限 |

### 来期の優先事項
1. ...

### 補足・参考データ
```
