# /press — プレスリリース・広報文作成

プレスリリース・スピーチ原稿・社内メール・SNS投稿を作成する。

## 使い方

```
/press [種別] [内容・情報]
種別: release（プレスリリース）| speech（スピーチ）| internal（社内メール）| sns（SNS）| crisis（危機対応）
```

## 手順

1. 種別と伝えたい内容を確認する
2. 対象読者とトーンを設定する
3. 5W1H を揃えてドラフトを作成する
4. 炎上リスク・誤解を招く表現がないかチェックする
5. ファイルに保存する

## トーン設定

| 種別 | トーン | 文字数 |
|------|--------|--------|
| release | 公式・中立 | 600〜1000字 |
| speech | 温かみ・ビジョン | 1000〜2000字 |
| internal | 丁寧・明確 | 300〜600字 |
| sns | 親しみ・簡潔 | 200〜400字 |
| crisis | 誠実・謝罪 | 400〜800字 |
