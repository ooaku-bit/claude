# /minutes — 議事録作成

会議の内容を入力すると「決定事項・宿題・次回予定」の議事録を即座に作成する。

## 使い方

```
/minutes [会議名・日時・参加者・議論内容をテキストで貼り付け]
```

## 手順

1. 入力されたテキストから会議情報を抽出する
2. 決定事項・未決事項・宿題をMECEに整理する
3. 宿題事項には担当者・期限を付与する（不明な場合は「要確認」と記載）
4. 議事録ファイルを `minutes_YYYYMMDD_[会議名].md` で保存する

## 出力フォーマット

```
## 議事録  YYYY-MM-DD  [会議名]
参加者: ...

### 決定事項
- 〇〇を実施する（担当: ）

### 未決事項（継続審議）
- 〇〇については次回決定

### 宿題事項
| 担当 | 内容 | 期限 |
|------|------|------|

### 次回会議
- 日時:
- 議題:
- 場所:
```
