# /research — 市場・競合調査

指定したテーマの市場・競合・トレンドを調査してレポートにまとめる。

## 使い方

```
/research [調査テーマ]
例: /research "国内SaaS市場の競合比較"
```

## 手順

1. 調査目的と対象範囲を確認する
2. WebSearch で最新情報を収集する
3. 3C / PEST フレームワークで整理する
4. 経営判断に使えるインサイトを3つ抽出する
5. レポートを `research_YYYYMMDD_[テーマ].md` で保存する

## 出力フォーマット

```
## 市場調査レポート  [テーマ]  YYYY-MM-DD

### エグゼクティブサマリー

### 市場概況
規模 / 成長率 / 主要プレイヤー

### 競合比較表
| 企業 | 強み | 弱み | 価格 | シェア |

### トレンド・機会
### 脅威・リスク

### 経営へのインサイト（3点）
①②③

### 情報源
```
