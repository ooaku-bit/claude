# /brief — 経営ブリーフィング作成

経営者が今日すべきことを1枚でまとめたブリーフィングシートを作成する。

## 手順

1. ユーザーから「本日の状況・優先事項・懸念点」をヒアリングする
2. 情報を以下のフォーマットで整理する
3. 「今日の最重要アクション」を3つ以内に絞る

## 出力フォーマット

```
## 経営ブリーフィング  YYYY-MM-DD（〇曜日）

### 今日の最重要アクション（Top 3）
1. 【期限: HH:MM】内容
2. ...
3. ...

### 本日の予定
| 時間 | 内容 | 場所/形式 |

### 確認が必要な事項
- ...

### 昨日からの持ち越し
- ...

### 今週のゴール
- ...
```
