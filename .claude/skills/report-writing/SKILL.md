# レポート作成スキル（Report Writing）

## レポート構成（標準テンプレート）

1. **エグゼクティブサマリー**（3行以内）
2. **現状・実績**（数値データを含む）
3. **課題・リスク**（優先度付き）
4. **推奨アクション**（担当・期限付き）
5. **参考資料・補足**

## 手順

1. 目的（誰に・何を伝えるか）を確認する
2. 利用可能な情報・データを `Read` で収集する
3. 構成を決めてドラフトを作成する
4. エグゼクティブサマリーを最後に書く（全体を読んだ上で3行にまとめる）
5. `Write` / `Edit` でファイルに出力する

## 品質基準

- A4換算で2〜4ページ以内に収める
- 数値は必ず単位・比較基準（前月比・前年比）を添える
- 結論は最初に書く
- 図表・箇条書きを積極的に活用する
