# リスク評価スキル（Risk Assessment）

## リスクカテゴリ

| カテゴリ | 内容 |
|---|---|
| 法務リスク | 契約・規制・知財・個人情報 |
| 財務リスク | キャッシュフロー・与信・為替 |
| 評判リスク | 炎上・メディア・SNS |
| 事業リスク | 競合・市場変化・依存先集中 |
| オペレーションリスク | システム障害・人材流出・災害 |

## 評価基準

- **発生確率**: 高（1年以内に発生しうる）/ 中（3年以内）/ 低（それ以上）
- **影響度**: 高（事業継続に影響）/ 中（業績に影響）/ 低（軽微）
- **優先度**: 発生確率×影響度で★1〜3で表示

## 手順

1. 対象（計画書・契約書・施策）を `Read` で確認する
2. 5カテゴリにわたってリスクを洗い出す
3. 各リスクを評価基準でスコアリングする
4. 優先度★★★のリスクに対策を立案する
5. レポートを `Write` で出力する

## 出力フォーマット

```
## リスク評価  [対象]  YYYY-MM-DD

### リスクマトリクス
| リスク | カテゴリ | 確率 | 影響 | 優先度 | 対策方針 |

### 要対応リスク（優先度★★★）

### 法的留意事項
※ 専門家（弁護士等）への確認を推奨する事項:
```
