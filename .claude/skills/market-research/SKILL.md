# 市場調査スキル（Market Research）

## 調査領域

1. **市場規模・成長率**: TAM / SAM / SOM の推定
2. **競合分析**: 主要競合のサービス・価格・強み・弱みを比較
3. **トレンド分析**: 業界の最新動向・技術変化・規制変化
4. **顧客インサイト**: ターゲット顧客のニーズ・課題・行動パターン

## 手順

1. 調査目的と対象市場を確認する
2. `WebSearch` で最新情報を収集する
3. `WebFetch` で信頼性の高い情報源の詳細を取得する
4. 情報を整理してフレームワーク（3C / PEST）に当てはめる
5. レポートを `Write` で出力する

## 出力フォーマット

```
## 市場調査レポート  [テーマ]  YYYY-MM-DD

### エグゼクティブサマリー

### 市場概況
- 規模: 〇〇円（YYYY年）
- 成長率: 年率〇〇%
- 主要プレイヤー:

### 競合比較
| 企業 | 強み | 弱み | 価格帯 | シェア |

### トレンド・機会
1. ...

### 脅威・リスク
1. ...

### 情報源
- ...（取得日: YYYY-MM-DD）
```
