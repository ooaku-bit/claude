# 会議準備スキル（Meeting Prep）

## 事前準備（会議前）

1. **アジェンダ作成**: 目的・議題・時間配分・参加者・決定事項の型を作成
2. **事前ブリーフィング**: 経営者が会議前に読む1枚ペーパーを作成
3. **想定Q&A**: 参加者から想定される質問と回答案を列挙

## 事後処理（会議後）

1. **議事録作成**: 決定事項・宿題事項・次回予定を整理
2. **フォローアップリスト**: 担当者・内容・期限をテーブルでまとめる

## アジェンダテンプレート

```
## 会議アジェンダ  YYYY-MM-DD HH:MM

目的: 〇〇を決定する

| 時間 | 議題 | 担当 | 目標 |
|------|------|------|------|
| 10分 | ...  | ...  | 情報共有 |
| 20分 | ...  | ...  | 意思決定 |

事前確認資料: ...
参加者: ...
```

## 議事録テンプレート

```
## 議事録  YYYY-MM-DD  [会議名]

### 決定事項
- 〇〇を✗✗する（担当: ）

### 宿題事項
| 担当 | 内容 | 期限 |

### 次回予定
日時 / 議題
```
