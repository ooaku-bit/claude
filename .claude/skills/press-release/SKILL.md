# 広報文書作成スキル（Press Release）

## 文書種別と要件

| 種別 | 文字数目安 | トーン | 必須要素 |
|---|---|---|---|
| プレスリリース | 600〜1000字 | 公式・中立 | 5W1H・会社概要・連絡先 |
| 経営者スピーチ | 1000〜2000字 | 温かみ・ビジョン | 開口・本論・結び |
| 全社メール | 300〜600字 | 丁寧・明確 | 目的・内容・アクション |
| SNS（LinkedIn）| 200〜400字 | 親しみ・学び | フック・本文・CTA |
| 危機対応文 | 400〜800字 | 誠実・謝罪 | 事実・謝罪・再発防止 |

## 手順

1. 文書種別・目的・対象読者を確認する
2. 必要な情報（事実・数値・背景）を `Read` で収集する
3. 構成を組み立ててドラフトを作成する
4. 炎上リスクのある表現がないかチェックする
5. `Write` / `Edit` でファイルに出力する

## 品質チェックリスト

- [ ] 5W1H が揃っているか
- [ ] 専門用語・略語を使っていないか
- [ ] ポジティブな言葉を選んでいるか
- [ ] 誰が読んでも誤解しない表現か
- [ ] 炎上リスクのある表現がないか
